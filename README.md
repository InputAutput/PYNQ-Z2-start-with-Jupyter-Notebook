# PYNQ-Z2 — Start with Jupyter Notebook

## EN

[![PYNQ-Z2 start with Jupyter Notebook](https://img.youtube.com/vi/viUCuciKcDw/maxresdefault.jpg)](https://youtu.be/viUCuciKcDw)

Video: https://youtu.be/viUCuciKcDw

In this video I walk you step by step through setting up the PYNQ-Z2 board to work with Jupyter Notebook: we download the official PYNQ image, flash it to a microSD card using Win32 Disk Imager, correctly set the power and boot jumpers, connect via Ethernet (RJ-45) through a router or directly to a PC, open Jupyter in the browser, and test the board with a simple example using a switch and an LED. This video is for anyone who wants a quick start with FPGA / Zynq, student labs, prototyping, and peripheral testing without a complex IDE — everything is done in Python right in the browser.

Win32 Disk Imager (tool for writing the image to microSD): https://win32diskimager.org/  
Guide for direct board-to-PC connection (Ethernet without a router): https://pynq.readthedocs.io/en/v2.3/appendix.html#assign-your-computer-a-static-ip
